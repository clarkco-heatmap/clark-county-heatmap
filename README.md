
# Clark County Building Age Heatmap

A live interactive heatmap showing **age of buildings in Clark County, WA**, using:

- Clark County GIS TaxlotsPublic dataset  
- ArcGIS Maps SDK for JavaScript 4.30  
- Reliable paged queries  
- True heatmap from centroids  

## Live URL (after publishing)
https://clarkco-heatmap.github.io/clark-county-heatmap/

## Features
- Older buildings show as **hot (red)**  
- Newer buildings show as **cold (blue)**  
- 100% reliable paging avoids server query limits  
- Works on GitHub Pages with no backend
